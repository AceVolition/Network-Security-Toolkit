# Network-Security-Scanner

A network scanning tool featuring:
- **Multi-threaded concurrent scanning** (8x faster than sequential)
- **Socket-based TCP scanning** with configurable timeouts  
- **Nmap integration** for advanced scan types (SYN, FIN, Xmas)
- **File-based batch processing** for IP/port lists
- **Comprehensive reporting** with CSV/JSON output
- **Modular architecture** supporting multiple scanning engines

## Key Features
- Concurrent scanning with multiprocessing Pool
- Integration with industry-standard Nmap
- Four scanning methodologies comparison
- Production-ready error handling and logging
